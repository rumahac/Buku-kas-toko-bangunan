Menggunakan metode regresi linear berdasarkan data 30 hari terakhir
Tampilan grafik historis vs prediksi (hingga 30 hari ke depan)
Statistik tren harian, estimasi saldo akhir, dan rekomendasi AI

**5. OCR (Optical Character Recognition) untuk Scan Struk**
Upload foto struk belanja, sistem akan membaca teks dan mengekstrak total jumlah secara otomatis
Mempercepat input transaksi, mengurangi kesalahan manual
Didukung oleh Tesseract.js di sisi klien
**6. Manajemen User (Khusus Admin)**
Tambah, edit, hapus user
Atur role (admin/operator/viewer) dan status (aktif/nonaktif)
Reset password user
Pantau last login setiap user
**7. Laporan Ekspor**
Ekspor data ke Excel (XLSX) dan PDF
Filter berdasarkan periode: hari ini, minggu ini, bulan ini, tahun ini, atau semua data
Tampilan tabel yang rapi dengan nomor halaman dan pencarian
**8. Antarmuka Modern & Responsif**
Desain minimalis dengan warna indigo, merah, hijau yang elegan
Tema gelap/terang yang dapat diganti dengan satu klik
Burger menu untuk tampilan mobile, memastikan kenyamanan di berbagai perangkat
Animasi halus pada setiap elemen
**9. Keamanan & Audit**
Semua aktivitas penting (login, logout, akses gagal) dicatat dalam security_logs
Proteksi CSRF pada setiap form
Rate limiting mencegah brute force
Validasi IP dan User-Agent untuk mencegah session hijacking
